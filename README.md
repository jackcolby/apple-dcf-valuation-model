# Apple Inc. Discounted Cash Flow (DCF) Valuation Model

A five-year discounted cash flow (DCF) valuation model built in Microsoft Excel to estimate the intrinsic enterprise value of Apple Inc. using analyst consensus forecasts and key valuation assumptions.

---

## Overview

This project forecasts Apple's operating performance and values the company using a Discounted Cash Flow (DCF) methodology.

The model projects revenue, operating income, taxes, free cash flow, and terminal value before discounting future cash flows back to present value using the Weighted Average Cost of Capital (WACC).

---

## Valuation Methodology

The valuation consists of the following components:

- Five-year revenue forecast
- Projected EBIT and NOPAT
- Free Cash Flow (FCF) forecast
- Discounted Cash Flow (DCF) analysis
- Terminal Value using the Gordon Growth Method
- Enterprise Value calculation
- WACC and Terminal Growth sensitivity analysis

---

## Key Assumptions

| Assumption | Value |
|------------|------:|
| WACC | 8.00% |
| Terminal Growth Rate | 2.50% |
| Tax Rate | 21.00% |
| EBIT Margin | 32.64% |

Revenue projections beyond FY2026 are based on analyst consensus estimates with gradually declining long-term growth assumptions to reflect Apple's mature business model.

## Valuation Summary

| Metric | Value ($B) |
|--------|-----------:|
| Present Value of Forecast FCF | 628.14 |
| Present Value of Terminal Value | 1,824.50 |
| Enterprise Value | **2,452.64** |
| Current Enterprise Value | 4,300.00 |
| Difference | -42.96% |
| Conclusion | **Overvalued** |

---

## Sensitivity Analysis

The model includes a two-variable sensitivity analysis examining the impact of changes in:

- Weighted Average Cost of Capital (WACC)
- Terminal Growth Rate

This analysis illustrates how enterprise value changes under different valuation assumptions.
